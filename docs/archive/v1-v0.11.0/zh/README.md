# ⚠️ 这是历史档案,不是当前文档

本目录(`docs/archive/v1-v0.11.0/zh/`)是 **v1 版本**的 hermes-agent 源码分析,基于 **hermes-agent v0.11.0**。

**当前正式文档在 [`docs/zh/`](../../../zh/)**,基于 hermes-agent **v0.14.0**(commit `3bace071b`,2026-05-24),共 14 章(00–13)。

## 为什么还留着

v1 的唯一使命是充当 v2 写作时的**深度基准**——「v2 必须覆盖 v1 维度的超集」(见 `docs/WORKFLOW.md`)。这个使命随 v2 全部完成 + 终审通过已经结束。保留它**仅作历史对照**:对比 v1(v0.11.0)与 v2(v0.14.0)能看出项目两个版本间的演进。

**不要再编辑、引用或基于本目录工作。** 任何「当前」的内容都应在 `docs/zh/`(顶层)。

## v1 → v2 结构变化

v1 是 12 章,v2 重组为 14 章。主要差异:

- v1 的 `06-gateway网关` → v2 拆为 `05-网关层` + `06-协议适配层`(ACP/MCP serve 独立)
- v1 的 `07-tui与web` → v2 的 `10-交互界面与运行模式`(并入运行模式)
- v1 无独立 Kanban 章 → v2 新增 `09-Kanban系统`
- v1 的 `05-插件系统` → v2 拆为 `07-插件框架` + `08-内置插件`
- v1 的 `10-批量运行与rl` → v2 的 `12-批量运行与轨迹生成`(RL 训练环境已在 v0.14.0 移出主仓,故改名聚焦数据生成)
- v1 的 `09-环境与部署` 内容并入 v2 的 `01-基础设施层` 与 `13-工程实践`

英文版历史档案见 [`en/`](../en/)（同一归档目录下）。
