# ⚠️ Historical Archive — Not Current Documentation

This directory (`docs/en-v1/`) is the **v1** hermes-agent source analysis, based on **hermes-agent v0.11.0**.

**The current documentation is in [`docs/en/`](../en/)** (once translated) and [`docs/zh/`](../zh/), based on hermes-agent **v0.14.0** (commit `3bace071b`, 2026-05-24), 14 chapters (00–13).

## Why it's kept

v1's only purpose was to serve as the **depth baseline** while writing v2 — "v2 must cover a superset of v1's dimensions" (see `docs/WORKFLOW.md`). That purpose ended when v2 was completed and passed final review. It is retained **for historical comparison only**: comparing v1 (v0.11.0) against v2 (v0.14.0) shows how the project evolved between the two versions.

**Do not edit, reference, or build on this directory.** All "current" content lives in `docs/en/` and `docs/zh/`.

See the Chinese archive note [`docs/zh-v1/README.md`](../zh-v1/README.md) for the v1→v2 chapter restructuring details.
